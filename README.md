Coffee Sales Dashboard (Excel)

An interactive Excel dashboard created to analyze coffee sales performance across products, customers, and regions. This project demonstrates data cleaning, analysis, and visualization skills using Microsoft Excel.

-- Project Overview

The Coffee Sales Dashboard provides insights into overall sales, profit, top customers, and country-wise performance. Interactive filters allow users to explore data dynamically and support data-driven decision-making.

-- Objectives

Analyze total sales and profit
Identify top-performing customers and products
Compare sales across countries
Visualize business performance using charts

-- Dataset

The dataset includes:
Orders Data: Order date, quantity, sales, profit
Customer Data: Customer name, country
Product Data: Coffee type, roast type, price
Data was cleaned and prepared before analysis to ensure accuracy.

--Tools Used

Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Data Cleaning & Formatting

-- Dashboard Features

Total sales and profit overview
Top 5 customers by sales
Country-wise sales distribution
Product-level performance analysis
Interactive slicers for filtering

-- Analysis Process

Data cleaning and preparation
Creation of pivot tables
Visualization using charts
Dashboard layout and design
Adding slicers for interactivity

-- Key Insights

Identified customers contributing highest revenue
Found top-selling and most profitable coffee products
Analyzed regional sales trends.

Conclusion

This project highlights practical Excel skills in data analysis and dashboard creation, making it suitable for portfolio and GitHub showcase.
